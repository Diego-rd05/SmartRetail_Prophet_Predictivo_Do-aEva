# SmartRetail_Prophet_Predictivo_Do-aEva
"Proyecto de pronóstico de demanda y análisis predictivo para Micromercado Doña Eva utilizando Meta Prophet."
📊 Optimización y Análisis de la Demanda: Micromercado Doña Eva
Este proyecto es un análisis integral de datos realizado para el Micromercado Doña Eva (Cochabamba, Bolivia), centrado en la auditoría de ventas, el impacto de los ciclos inflacionarios y la proyección de demanda mediante el uso de inteligencia artificial.

Objetivo: Auditar la consistencia operativa entre 2023 y 2026 e implementar un modelo de pronóstico (Prophet de Meta) adaptado a la estacionalidad local para optimizar la gestión de inventarios.

🛠️ Tecnologías y Herramientas
Lenguaje: Python 3.x

Análisis de Datos: pandas, numpy

Visualización: matplotlib, seaborn

Modelado Predictivo: prophet (Meta)

Entorno: Google Colab

📈 Hallazgos Principales
Auditoría de Calidad: Se procesaron más de 79,000 transacciones con una integridad del 100%.

Análisis Inflacionario: Se identificó que, a pesar de los picos inflacionarios detectados entre 2024 y 2025, el negocio logró mantener un margen de ganancia neto estable (promedio del 21%), demostrando una excelente capacidad de adaptación de precios.

Estructura del Ticket: El negocio se sostiene sobre una estrategia de alta rotación con un ticket promedio de 10.42 Bs.
